These are pre-defined functions that are supposed to make programming easier by removing the need to code mundane tasks.

## Basic

| Name    | Usage                               |
| ------- | ----------------------------------- |
| display | Displays string in terminal.        |
| input   | Takes in values typed after prompt. |
| int     | Converts variable into an integer.  |
| float   | Converts variable into an float.    |
| str     | Converts variable into an string.   |
## Lists
Below is a table of functions that can be used on lists

| Name         | Usage                                           |
| ------------ | ----------------------------------------------- |
| min          | Returns the minimum value in a list.            |
| max          | Returns the maximum value in a list.            |
| sort         | Returns a sorted list.                          |
| mean         | Returns the mean of the list.                   |
| median       | Returns the median of the list.                 |
| mode         | Returns the mode of the list.                   |
| total        | Returns the sum of all the values of the list.  |
| random_item  | Returns a random element of the list.           |
| display_list | Returns a string of the list in a nicer format. |

## Maths
| Name       | Usage                                   |
| ---------- | --------------------------------------- |
| absolute   | Returns the absolute value of a number. |
| random_int | Returns a random integer in the range.  |
| sqrt       | Returns the sqrt of a number.           |

## Other
| Name         | Usage                                                                                              |
| ------------ | -------------------------------------------------------------------------------------------------- |
| clear_screen | Clears the terminal. (Optional - just makes things look nicer!)                                    |
| is_int       | Returns true if variable is an integer and false otherwise.                                        |
| is_float     | Returns true if variable is a float and false otherwise.                                           |
| is_str       | Returns true if variable is a string and false otherwise.                                          |
| is_upper     | Returns true if variable is an uppercase character or string is ALL uppercase and false otherwise. |
| is_lower     | Returns true if variable is a lowercase character or string is ALL lowercase and false otherwise.  |

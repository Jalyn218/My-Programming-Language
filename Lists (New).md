This contains all methods for lists which are all basically the same as Python.

```text
var my_list = [1,3,4,5]
```
## Retrieving elements
```text
display(my_list.get(1)) >> 3 
```

## Adding elements
```text
my_list.add(60)
display(my_list) >> [1,3,4,5,60]
```

## Removing elements
```text
my_list.remove(2)
display(my_list) >> [1,3,5]
```

## Getting the position of an element
```text
var position = my_list.get_position(4)
display(position) >> 2
```

## Clearing a list
```text
my_list.clear()
display(my_list) >> []
```
